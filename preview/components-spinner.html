Usage

SizeUse when
16 pxInline within body text or a table cell. Use sparingly — the mark becomes very small.
32 pxButton loading state, list-level loading, card skeleton. This is the default.
64 pxSection or panel loading takeover — e.g. club feed refreshing.
128 pxFull-screen page-level loading only. Above 128 px the stroke starts to look thin.
Rule
Do Always pair with "Loading…" copy (Sentence case, secondary color) in page-level contexts.
Do Use alt="Loading" on the <img> or aria-label="Loading" on inline SVG.
Don't Use more than one spinner per screen. Multiple spinners signal a broken loading architecture.
Don't Change the color. The spinner always uses --kluvs-primary (#D16D30). It reads on both surfaces.
Don't Override the animation duration. The 4 s rhythm is intentional.
